1.6. 대상 표기#

1.6.1. 공통#

n // 0 포함 자연수
-n // 음의 정수
1/0 // 무한대 (Infinity)
true/false // 불리언
a1, a2, a3 ...a // 지정 값
index // 인덱스
target // 지정 인덱스
start // 시작 인덱스
end // 끝 인덱스
start -> end // 차례로
end -> start // 역순으로
r/ // 정규표현식
r1/, r2/, ...r/ // 특정 정규 표현식

1.6.2. 문자열#

str,
'str'[start]: 's'
'str'[end]: 'r'
'a' // 지정 문자열
`{t}` // 템플릿 리터럴

1.6.3. 배열#

arr, // []
[item1, item2, ...item]: arr[index],
arr[start]: item1,
arr[end]: ...item,

1.6.4. 객체#
