3. ascii num 변경#

str.charCodeAt(); // ascii num으로 변경

4. str 정규표현식 사용하여 replace로 교체#

5. arr 배열 내 문자열 정렬#

arr.sort((a, b) => {
  return a[0].charCodeAt() - b[0].charCodeAt();
});

6. str 정규표현식 사용하여 알파벳 소문자만 남기고 숫자, 기호 등 제거#
